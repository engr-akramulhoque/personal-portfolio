## Installation Guide üßë‚Äçüíª

### Using Git and Github

### Clone the Repository

```bash
  git clone https://github.com/engr-akramulhoque/personal-portfolio.git
```

### Install required packages

```bash
  cd personal-portfolio
```

```bash
  npm install
```

### Run server

```bash
  npm run dev
```
